Телегам-бот Emigrant Lives.

Демонстрационный бот, который предоставляет информацию о городах с большими русскоязычными общинами. Идеально подходит для потенциальных переселенцев или просто любопытных! Бот использует платформу обработки данных TinyGods, получая и агрегируя информацию из множества источников, включая публичные API и веб-сайты.
